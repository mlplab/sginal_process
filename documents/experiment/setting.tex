\section{実験環境}

以下に本レポートにて行った実験環境について述べる．

\begin{itemize}
  \item OS: Windows10
  \item プログラム言語: Python 3.6
  \item ライブラリ
  \begin{itemize}
    \item Numpy: 数値計算ライブラリ
    \item OpenCV: 画像処理ライブラリ
    \item MatPlotLib: 画像出力ライブラリ
  \end{itemize}
\end{itemize}
